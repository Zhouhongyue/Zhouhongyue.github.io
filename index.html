实验二 git使用

标签（空格分隔）： 未分类

##一、实验目的
1.掌握git使用
2.掌握使用Github作为远程仓库

##二、实验任务
1.练习本地git操作
2.用SSH同步本地与Github仓库

##三、实验步骤与结果
1.本地操作log截图
2.Github上commits截图
步骤如下：

①下载安装GitHub桌面版
②打开GitHub桌面版，点击左上角的“+”号，create一个新的repository，我将此命名为“test”

③打开我保存的“test”文件地址，并在此目录下新建一个名为“hello”的txt文件，随意添加文本内容

④打开GitHub桌面版，查看new changes，添加描述并commit to master，再点击右上角的publish，将此同步到GitHub中

⑤点击图标，create new branch，自行命名，publish后可在网页上查看


##四、实验小结
Git是一个软件，是一个版本分布式管理工具。
GitHub是一个网站，用来托管各种项目

实验一 在Github上用Markdown创建技术博客

标签（空格分隔）： 未分类

##一、实验目的

  1. 熟悉Github网站
  2. 掌握Markdown语法
    ##二、实验任务
    1.注册Github账号
    2.创建Github Pages repository
    3.发布第一篇博客
    4.学习Markdown
    5.利用Hexo生成博客并部署到Github
    ##三、实验步骤与结果
    （一） 1.安装git
    2.安装node
    3.设置npm镜像为淘宝
    npm config set registry https://registry.npm.taobao.org
    4.安装hexo
    npm install -g hexo-cli
    5.初始化站点
    hexo init myhexo
    cd myhexo
    npm install
    6.浏览本地站点
    hexo server
    7.在Github上部署hexo
    8.新建博客
    hexo new testMyBlog
    9.生成博客
    hexo generate
    10.发布到服务器
    hexo deploy
    （二）fork别人的博客，修改并提交请求
    1.Fork他人的仓库
    2.Clone我的仓库
    3.更新，提交更改
    4.将更改push到GitHub仓库并给对方发送一个pull request
    ##四.实验小结
    注意选好github的版式。
